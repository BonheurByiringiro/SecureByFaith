# Secure By Faith Portfolio

> Professional portfolio showcasing expertise in AI Security, Machine Learning, and Cybersecurity - Guided by faith, powered by innovation

##  Features

- **Modern Next.js 14** with App Router
- **TypeScript** for type safety
- **Tailwind CSS** for responsive design
- **Cyberpunk-inspired** dark theme with neon accents
- **Smooth animations** with Framer Motion
- **SEO optimized** with Next.js metadata
- **Mobile responsive** design
- **Fast performance** optimized for Vercel

##  Sections

1. **Hero** - Animated introduction with typewriter effect
2. **About** - Background in AI and cybersecurity
3. **Skills** - AI/ML, Security Tools, Cloud & DevSecOps
4. **Projects** - Featured work in AI security
5. **Blog** - Insights on security and AI
6. **Contact** - Social links and contact info
